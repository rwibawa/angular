# angularJS
angularJS exercise

> Requirements:
> * NodeJS
> * NPM

## 1. Setup
```shell
  $  mkdir express_02
  $  cd angularJS_02
  $  npm install http-server -g
  $  http-server -p 9003
```
Then open:
* [the website](http://localhost:9003/index.html)
* [the website](http://localhost:9003/index2.html)
